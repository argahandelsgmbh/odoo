Website Floating WhatsApp Button

Add a professional floating WhatsApp chat button to your Odoo website in minutes. Engage with your customers directly through WhatsApp with minimal configuration.

Super lightweight, fully customizable, and easy to manage from your Website Settings.

Module Features

Floating Button

Eye-catching floating button with smooth pulse animation that attracts customer attention.

Direct Chat Link

Click to open WhatsApp chat with pre-filled message. No manual typing required by customers.

Dynamic Phone Number

Easily configure your WhatsApp business number with country code from settings.

Custom Message

Set default greeting message that appears when visitors click the button.

Position Control

Choose button position on left or right side of the screen as per your preference.

Size Options

Select from Small (50px), Medium (60px), or Large (70px) button sizes.

Enable/Disable

Turn the button ON or OFF from Website Settings with a single click.

Hover Tooltip

Customizable tooltip text that appears when users hover over the button.

Mobile Responsive

Fully optimized for all devices - desktop, tablet, and mobile phones.

Lightweight

Pure CSS and JavaScript - no heavy dependencies or external libraries.

Smooth Animation

Beautiful entrance animation and pulse effect to grab customer attention.

Accessibility

Respects user preferences including reduced motion and dark mode.

How to Configure

Step 1: Install Module

Go to Apps, search for "Website Floating WhatsApp Button" and click Install.

Step 2: Enable Button

Navigate to Website Settings and enable the WhatsApp button toggle.

Step 3: Add Phone Number

Enter your WhatsApp business number with country code (e.g., +1234567890).

Step 4: Customize Message

Set default greeting message and tooltip text as per your preference.

Step 5: Choose Position & Size

Select button position (left/right) and size (small/medium/large) from settings.

Step 6: Save & Enjoy

Save settings and your WhatsApp button is now live on your website.
